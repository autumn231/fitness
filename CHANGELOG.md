# 更新日志

本文件记录每个发布版本的更新内容。GitHub Actions 发布时会自动从中提取对应版本说明填入 Release Notes。

## v2.3

- 总用时计时支持三位小时显示（`hh:mm:ss`），可支撑 3-4 小时训练
- 训练执行新增暂停 / 继续功能，进入页面默认不计时
- 训练改为导航栏独立 Tab（5 Tab：首页 / 分类 / 训练 / 日历 / 我的），可在训练页切换不同计划
- 计划详情页移除"开始训练"按钮，专注计划管理
- 新增日历功能：月历视图、为指定日期安排训练计划、已安排日期圆点标记、"即将到来"列表

## v2.2

- 图标改回简洁哑铃风格，并保留安全边距不再贴边
- 新增训练执行页，入口位于计划详情页 ▶ 按钮，包含：
  - 总用时计时（进入即开始）
  - 组间歇倒计时（30s / 60s / 90s / 120s 四档预设 + 开始/暂停/重置）
  - 训练清单待办，点击勾选完成动作并显示进度

## v2.1

- 图标重绘为像素风格手握哑铃设计，四周留白不贴边
- 修复进入主页时不会自动加载数据、需切换页面后才显示的问题
- 作者信息从设置页合并到关于页
- 修复点击首页"开始训练""动作分类"等快捷入口跳转后，底部导航栏点击主页无响应的问题（导航状态保存逻辑修正）

## v2.0

UI 全面现代化重构。

- Material 3 Expressive 色彩体系（活力橙 + 珊瑚 + 青绿），完整浅深色 surfaceContainer 层级
- 14 级 Typography 现代字体节奏
- 5 级圆角体系 + 卡片 / Chip / Pill / 图片预设形状
- 页面切换动画：子页滑动 + 淡入 + 缩放（320ms），Tab 主页纯淡入淡出
- 现代化组件：Hero 卡片、骨架屏、空状态、悬浮 FAB、沉浸式详情页（视差滚动 GIF）
- 新增清除历史功能：收藏页双 Tab（收藏 / 最近浏览）+ 设置页清空入口，均带确认对话框
- minSdk 提升至 31（Android 12），启用预测式返回动画
- 沉浸式透明状态栏 / 导航栏

## v1.0

首个发布版本。

- 离线健身动作指导应用，无需联网
- 内置 1300+ 动作中文指导 + GIF 动图演示
- 按身体部位 / 目标肌群 / 器械多维度分类浏览
- 关键词快速搜索
- 收藏喜欢的动作，自动记录最近浏览历史
- 自定义训练计划：自由添加动作并设置组数与次数
- 支持浅色 / 深色 / 跟随系统主题
